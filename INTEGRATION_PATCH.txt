PolyAlpha Terminal integration into your existing PolyScalpBot

1) Copy these files into your current repo:

bot/smart_money.py
bot/alpha_store.py
bot/alpha_handlers.py

2) Open bot/handlers.py

At the top, add:

from bot.alpha_handlers import register_alpha_handlers

At the bottom inside register_handlers(app), add this line:

register_alpha_handlers(app)

IMPORTANT: Put it before/after your other app.add_handler lines, but still inside register_handlers(app).

3) No new requirements needed if your bot already has:

python-telegram-bot
httpx

4) Railway env stays mostly same:

TELEGRAM_TOKEN=your_token
DB_PATH=bot.db
ADMIN_USER_IDS=your_telegram_id_optional

5) Start bot, then in Telegram:

/alpha
/alpha_addwallet 0xWALLET label
/alpha_wallets
/topwallets
/consensus
/mywallet 0xYOURWALLET
/compare
/terminal

6) Safety:

/sell is intentionally disabled. Do not enable real trading until we add:
- CLOB API auth
- sell preview
- confirmation code
- max slippage
- max size
- emergency kill switch
